第一章 機器學習基礎

機器學習是人工智慧的一個分支。它讓電腦能夠從數據中學習。

1.1 監督式學習

監督式學習使用標記數據進行訓練。

  • 分類：預測類別標籤
  • 回歸：預測連續數值

1.2 非監督式學習

非監督式學習使用未標記數據。

  • 聚類：將相似數據分組
  • 降維：減少特徵數量

第二章 線性回歸

線性回歸是最基礎的回歸演算法。

2.1 模型定義

y = wx + b

2.2 損失函數

MSE = (1/n) * Σ(y_i - ŷ_i)²

2.3 梯度下降

w = w - α * ∂MSE/∂w
b = b - α * ∂MSE/∂b

第四章 神經網路

神經網路由多層神經元組成。

4.1 感知機

def perceptron(x, w, b):
    z = sum(x_i * w_i for x_i, w_i in zip(x, w)) + b
    return 1 if z > 0 else 0

4.2 激活函數

  • Sigmoid：1 / (1 + e⁻ˣ)
  • ReLU：max(0, x)
  • Tanh：(eˣ - e⁻ˣ) / (eˣ + e⁻ˣ)
